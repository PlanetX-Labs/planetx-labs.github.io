# Online Document Editing

DASSET provides a third-party online document editing tool as an extension application. With this tool, you can view and edit documents stored on the DASSET device directly.
*Supported formats include major Microsoft Office file types (DOC, DOCX, XLS, XLSX, PPT, PPTX, Visio, MS Publisher), standardized open document formats (ODT, ODS, ODP), and PDF.*

### Installing the Online Document Editing Tool
1.  In the third-party applications, select Online Document Editing Install.
![](../pc-images/image237.png)
2.  Wait for the installation to complete. The tool will be automatically installed on the DASSET device. You may also use the dropdown menu to disable or uninstall the tool.
3.  Once installed, you can open and edit documents stored on the DASSET device directly.
:::tip
Document preview and editing are performed on the DASSET device itself, without downloading files to your computer.
:::
![](../pc-images/image238.png)

### Online Document Preview and Editing
You can select any document stored on the DASSET device and choose Online Edit to preview or edit its content. *All edits are saved directly on the DASSET device.*
Example using a Word document:
1.  Select the document and choose Online Edit.
![](../pc-images/image239.png)
2.  The online editing tool will open the Word document, allowing you to view and edit it.
![](../pc-images/image240.png)
*For more details on using the online document editing tool, refer to its built-in help documentation.*
